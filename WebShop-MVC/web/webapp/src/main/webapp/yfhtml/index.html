成都天道筑亿丰科技有限公司

成都市天道筑亿丰科技有限公司成立于2017年，注册资本1000万元，主要从事不良资产管理，改造，处置业务。

公司团队核心成员是由一批长期从事不良资产收购、改造、处置的业内人士组成，而该团队从2004年成立至今，已成功完成了多个不良资产项目的改造及处置，但是进入2016年后，整个资管市场发生了根本性的变化，为了更好地适应市场，让公司得到发展，我们又重新聚集了一批从事数据分析、金融行家的人才，我们励志成为不良资产处置的行家。

经过近20年的发展，中国的城市化率由1998年的30.4%发展到2016年底的57.35%，平均每年以近1.5个百分点递增。城市化的快速进程也伴随着房地产行业的高速发展，见证了中国经济的腾飞，但也埋下了不良资产问题的种子。

展望未来，房地产行业定会完成增量开发到存量优化的转变。公司以不断学习、吸纳人才、广泛合作的心态，为不良资产的处置提供专业的方案和通道，让居者宜居、商者宜商。

公司秉承“求真务实”的企业文化，重视每一个员工的职业发展和专业培养，缔造企业与员工共同发展的平台，实现成为“不良资产处置专家”的愿景。
